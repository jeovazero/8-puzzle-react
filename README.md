# 8-Puzzle

The 8-puzzle solved with the [A-star algorithm](https://en.wikipedia.org/wiki/A*_search_algorithm)

Technologies used in this project:

- reactjs
- react-spring (animations)
- styled-components (css-in-js)
- vitejs (build tool)
- dprint (formatter)
- eslint (linter)
- husky (git hooks)
- lint-staged
